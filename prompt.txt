당신은 크롬 확장 프로그램(MV3) 개발자다. 아래 요구사항을 만족하는 최소/명확/견고한 코드를 생성하고, 필요 파일을 전부 만들어라. 불필요한 설명은 최소화하고, 실행 방법만 간단히 문서화한다.

[제품 목표]
- 사용자가 업로드/붙여넣기한 디자인 이미지(PNG)와 현재 탭 화면을 비교하여 시각적 차이를 즉시 하이라이트하는 도구
- 기본 모드: Overlay(겹쳐보기), Diff(차이 영역 빨간색 표시)

[핵심 요구사항]
1) MV3 기반, 퍼미션 최소: ["activeTab","scripting","tabs"] (클립보드 붙여넣기 지원은 브라우저 기본 paste 이벤트로 처리. navigator.clipboard.read 미필수)
2) 팝업(UI)
   - 붙여넣기 지원(⌘/Ctrl+V로 PNG 수신)
   - 슬라이더: 임계값(0~1), 오버레이 불투명도(0~1)
   - “현재 화면과 비교” 버튼
3) 비교 처리(콘텐츠 스크립트)
   - 오버레이 루트는 투명 배경, 페이지를 가리지 않게
   - 캔버스 3장: 디자인, Diff
   - 스케일 기준: 디자인을 맞춤(뷰포트 대비 화면에 맞춤/1:1 토글, 디폴트 360x800)
   - 드래그로 미세 정렬(스케일 고려)
   - 모드 토글: Overlay/Diff
   - Diff는 픽셀 밝기 차이 기반(간단하고 빠른 방식), 빨간색 마스크
4) UX 제약
   - 오버레이 닫기 버튼 제공, ESC로도 종료
   - DOM 충돌 방지: 최상위 z-index, Shadow DOM 또는 고유 prefix 클래스 사용
5) 성능/안정
   - 대형 이미지도 끊김 최소화(OffscreenCanvas/transferControlToOffscreen 지원 가능 시 사용, 미지원이면 기본 Canvas)
   - 에러 핸들링(클립보드/파일 미선택/캡쳐 실패 시 안내)
6) 산출물
   - manifest.json, popup.html/js, content.js, styles(필요 시), README.md
   - 아이콘은 임시 플레이스홀더 또는 생략 가능
7) 빌드 없이 동작할 수 있는 순수 JS/HTML/CSS(번들러 불필요)
8) README.md에 설치/사용법 명확히: chrome://extensions → 개발자 모드 → 폴더 로드 → 사용

[수용 기준(간단 테스트)]
- Figma에서 Copy as PNG 후 팝업에 ⌘/Ctrl+V → “현재 화면과 비교” → Overlay로 겹쳐 보임
- Diff 모드에서 차이 영역 빨간색 하이라이트
- 드래그 정렬이 붙여넣기 한 화면을 이동시키며 자연스럽게 작동
- ESC 또는 닫기 버튼으로 깔끔히 종료

이제 위 요구사항을 만족하는 파일들을 생성해라.
